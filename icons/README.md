# 图标文件说明

请在此目录下添加以下图标文件：

- `icon16.png` - 16x16 像素，用于扩展管理页面
- `icon48.png` - 48x48 像素，用于扩展管理页面
- `icon128.png` - 128x128 像素，用于 Chrome 网上应用店

## 临时解决方案

如果暂时没有图标文件，可以：

1. 使用在线工具生成简单的图标
2. 使用任何图片编辑软件创建 16x16、48x48、128x128 的 PNG 图片
3. 或者暂时注释掉 `manifest.json` 中的 `icons` 配置（插件仍可正常工作，但会显示默认图标）

## 图标设计建议

- 使用简洁的设计，在小尺寸下也能清晰识别
- 建议使用与"分组"或"标签"相关的图标
- 可以使用渐变色（如紫色/蓝色）与插件 UI 风格保持一致
