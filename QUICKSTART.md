# 快速开始指南

## 安装步骤

1. **准备图标文件**（可选但推荐）
   - 在 `icons/` 目录下添加三个图标文件：
     - `icon16.png` (16x16 像素)
     - `icon48.png` (48x48 像素)  
     - `icon128.png` (128x128 像素)
   - 如果没有图标，插件仍可正常工作，但会显示默认图标

2. **加载插件到 Chrome**
   - 打开 Chrome 浏览器
   - 访问 `chrome://extensions/`
   - 开启右上角的"开发者模式"开关
   - 点击"加载已解压的扩展程序"
   - 选择本项目文件夹

3. **配置 API Key**
   - 点击插件图标，选择"打开设置"
   - 或右键点击插件图标，选择"选项"
   - 输入 DeepSeek API Key
   - 点击"保存设置"

## 使用方法

### 方式一：插件图标右键
1. 在浏览器工具栏的插件图标上右键
2. 选择"AI分组"
3. 等待分组完成

### 方式二：插件弹窗（推荐）
1. 点击浏览器工具栏中的插件图标
2. 点击"立即分组"按钮
3. 等待分组完成

## 常见问题

### Q: 右键菜单中没有"AI分组"选项？
A: 请确保：
- 插件已正确安装并启用
- 在页面内容区域右键，或在插件图标上右键
- 注意：Chrome API 不支持在标签页上直接右键显示自定义菜单
- 刷新页面后重试

### Q: 分组失败？
A: 请检查：
- API Key 是否正确配置
- 网络连接是否正常
- 浏览器控制台是否有错误信息（按 F12 打开开发者工具）

### Q: 没有未分组的标签页？
A: 如果所有标签页都已分组，插件不会执行任何操作。这是正常行为。

## 技术说明

### 关于右键菜单的实现

**重要**：Chrome API **不支持**在标签页上直接右键显示自定义菜单项（`contexts: ['tab']` 不存在）。这是 Chrome 扩展 API 的限制。

本插件提供了以下替代方案：
- **页面右键菜单**：使用 `contexts: ['page']`，在页面内容区域右键时显示
- **插件图标右键**：使用 `contexts: ['action']`，在插件图标上右键时显示
- **插件弹窗**：点击插件图标，使用"立即分组"按钮（推荐方式）
