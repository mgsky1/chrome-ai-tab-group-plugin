# AI标签分组 Chrome 插件

一个智能的 Chrome 浏览器插件，可以根据网页标题自动对标签页进行分组。

## 注意
该项目基本骨架由AI实现，使用工具为[cursor](https://cursor.com/cn)

prompt
```
帮我制作一个Chrome插件，功能是能根据标签网页标题智能进行网页标签分组，即同类型的网页合并为一组。以下是功能详细描述：
1、希望能够在鼠标在浏览器标签栏右键时，在弹出的菜单中增加一项：「AI分组」。如果这个需求做不到，请告诉我做不到的原因，并且提供一个代替方案
2、当用户点击「AI分组」时，插件会分析当前打开的所有标签页的标题，并根据标题内容智能分组。
3、如果在已经分组的情况下，有新的未分组的标签页，只需要对未分组的标签页分组即可，分组时需要考虑其是否在已有分组中，如果都不在，再新建分组
4、如果没有未分组的标签，那么不进行分组
5、插件的AI供应商要可以配置，默认先接入DeepSeek。你写代码时需要考虑一下扩展性，以后接入新的AI供应商时能够方便地进行修改。
6、插件需要有一个简单的设置界面，用户可以在其中选择AI供应商。
7、你需要自己编写prompt，大概的意思是根据网页标题对标签进行分类，然后相同类别的合并为一组
```

## 功能特性

- 🤖 **AI智能分组**：使用 AI 分析网页标题，自动将相似主题的标签页归为一组
- 🎯 **右键菜单**：在标签页上右键即可快速分组
- 🔄 **增量分组**：只对未分组的标签页进行分组，已存在的分组会被保留
- ⚙️ **可配置AI供应商**：支持配置不同的 AI 服务提供商（默认 DeepSeek）
- 🎨 **美观界面**：现代化的设置界面和弹出窗口

## 安装方法

1. 下载或克隆此项目
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

## 使用说明

### 首次使用

1. 点击插件图标，选择"打开设置"
2. 在设置页面中：
   - 选择 AI 供应商（默认 DeepSeek）
   - 输入API信息
3. 点击"保存设置"

### 分组标签页

有三种方式可以触发分组：

1. **页面右键菜单**：在页面内容区域右键，选择"AI分组"
2. **插件图标右键**：在插件图标上右键，选择"AI分组"
3. **插件弹窗**：点击插件图标，然后点击"立即分组"

**注意**：由于 Chrome API 的限制，无法直接在标签页上右键显示自定义菜单。我们提供了上述替代方案。

### 分组规则

- 插件会自动分析所有未分组的标签页标题
- 如果标签页可以归入已有分组，会被添加到该分组
- 如果无法归入已有分组，会创建新的分组
- 如果没有未分组的标签页，不会执行任何操作

## 扩展新的AI供应商

要添加新的 AI 供应商，主要步骤
- `options.html`添加新的供应商
- `options.js`实现API数据存储
- `background.js`实现大模型调用

如果供应商API支持的是OpenAI标准，那么流程大同小异，参考现有实现即可。如果有其他协议实现，需要自行适配。

## 注意事项

1. **图标文件**：需要自行创建 `icons/` 目录并添加图标文件（16x16, 48x48, 128x128 像素）
2. **API Key**：请妥善保管您的 API Key，不要泄露给他人
3. **网络连接**：插件需要网络连接才能调用 AI API

## 开发说明

### 关于右键菜单

本插件提供了以下替代方案：
- **页面右键菜单**：使用 `contexts: ['page']`，在页面内容区域右键时显示"AI分组"选项
- **插件图标右键**：使用 `contexts: ['action']`，在插件图标上右键时显示"AI分组"选项
- **插件弹窗**：点击插件图标，使用"立即分组"按钮

如果需要在标签栏空白处右键也能显示菜单，可以考虑使用其他方案，比如：
- 使用快捷键触发分组（需要添加 commands API）
- 在标签栏添加自定义按钮（需要更复杂的实现）
